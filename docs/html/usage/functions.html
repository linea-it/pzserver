Connects to the Photo-z Server’s database and download a compressed zip file containing all the data and metadata of a given data product.

Parameters:

  • save_in (str) – location where the file will be saved

get_product(product_id=None, fmt=None)

Fetches the data product contents to local.

Connects to the Photo-z Server’s database and fetches the tabular data stored as registered data product.

Parameters:

  • product_id (str or int) – data product unique identifier (product id number or internal name)

  • fmt (str) – output table format ‘pandas’ -> pandas.DataFrame ‘astropy’ -> astropy.Table None (default) -> object class from catalog.py

Returns:

SpeczCatalog or TrainingSet (pandas.DataFrame extensions) object, or pure pandas.DataFrame, or astropy.Table
